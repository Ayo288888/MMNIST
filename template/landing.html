The Process

From raw traffic to actionable intelligence

Four automated steps transform raw network data into a classified threat report in real time.

1

Input Network Data

Select network feature values such as source bytes, duration, service type and connection counts from guided dropdowns drawn from real traffic profiles.

2

ML Pre-processing

Input features are standardized by a fitted scaler, normalizing each value against the training distribution before classification.

3

Attack Classification

The decision tree model evaluates scaled features and assigns the connection to one of 23 categories, from normal traffic to rootkit activity.

4

Alert and Log

Malicious detections are stored with severity ratings, added to the alerts dashboard, and surfaced in system logs for analyst review.
